    .JWPG Portfolio Web Site

    작업물의 결과를 효율적으로 전시하기 위해 .JWPG 포트폴리오 웹사이트를 만들었습니다.

    TYPOGRAPHY

    웹사이트 서체는 영어, 한글, 일본어가 모두 포함되어 있는 Noto Sans로 같은 스타일을 공유합니다.

    COLOR

    색상은 일관되고 의미있는 방식으로 UI 요소 및 구성 요소에 적용됩니다.

    Product Architecture

    LAYOUT

    반응형 레이아웃 그리드는 화면 크기와 방향에 맞게 조정되어 레이아웃간에 일관성을 보장합니다.

    SCROLL

    방문자의 편의를 고려해 직관적으로 볼 수 있게 디자인하였습니다.
    동일한 계층안에 컨텐츠를 구성해, 스크롤만으로도 컨텐츠 이동이 가능합니다.

    Components

    NAVIGATION

    네비게이션은 동일한 계층 수준에서 화면 간 이동 하며, 디바이스에 따라 각각의 방법을 통해 활성화됩니다.
    ① Desktop - 동일한 계층으로 스크롤 되어 이동합니다.
    ② Mobile - 하위 요소(네비게이션)가 상위 요소 위로 확장이 됩니다.

    COMPONENTS REFLOW

    UI는 기기별로 Reflow 되도록 레이아웃을 변경됩니다.
    Desktop의 수평 목록은 Mobile 화면에서 수직 목록으로 Reflow 됩니다.

    COMPONENTS TRANSITIONS

    탐색 전환은 사용자가 홈 화면에서 컨텐츠 화면으로 이동하는 등 화면 사이를 이동할 때 발생합니다.
    ① Desktop - Ajax 팝업을 통해 하위 요소(Components)가 상위 요소 위로 팝업이 됩니다.
    ② Mobile - 하위 요소(Components)가 상위 요소 위로 확장이 됩니다.
