    LG Wi-Fi Speaker

    LG Wi-Fi Speakerは、 LG Wi-Fiスピーカー専用のアプリケーションによって、 モバイル機器の音楽を聞いたり、 スピーカーの機能を設定することができます。

    ABOUT

    情報(スピーカー、音楽ファイルの探索) 機能(スピーカー、音楽ファイル管理、再生)、両方を提供します。

    PRODUCT ARCHITECTURE

    おしゃれで表現力に優れていて、ユーザーに最適化されたビジュアルとディテールなモーションでデザインされました。

    COLOR SYSTEM

    ICONOGRAPHY

    ユーザーがコンテンツにアクセスしやすいように、 シンプルに認識できるアイコンを使用しました。

    Screen Design

    ON BOARDING

    ユーザーの旅を案内すると同時に楽しさを与えます。

    SETUP GUIDE SCREEN

    Setup Guide構成要素に アイコン・イラストを配置し、理解を深めます。

    MANAGEMENT SCREEN

    下位情報を含めるスピーカーの区分及び管理を最上位に置いて、 音楽の再生・管理を下位セクションに分けます。

    MUSIC PLAYER SCREEN

    音楽の再生と同時に雰囲気に合うグラフィックを提供するため、 アルバムのカラーによってフレキシブルに変化するカラーテーマを取り入れました。

    PROTOTYPE

    カラーの変化とインタラクションの理解を 深めるためのプロトタイプ画面です。
