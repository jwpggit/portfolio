    .JWPG Portfolio Web Site

    作業の結果物を効率的に展示するため、 JWPG ポートフォリオ・ウェブサイトを作りました。

    TYPOGRAPHY

    ウェブサイトのフォントは英語、ハングル、日本語に対応出来るNoto Sansで、同じスタイルを共有します。

    COLOR

    カラーは統一されていて、意味のある方式でUI要素及び構成要素に適用されます。

    Product Architecture

    LAYOUT

    反応型レイアウトグリードは画面の大きさや方向に合わせて調整され、 レイアウト間の統一性を保障します。

    SCROLL

    来訪者の利便性のため、直感的に見えるようにデザインしました。
    同じ階層の中にコンテンツを入れ、スクロールだけでコンテンツを行き来できます。

    Components

    NAVIGATION

    ナビゲーションは同じ階層レベルで画面同志で移動し、 デバイスによってそれぞれの方法で活性化されます。
    ① Desktop - 同じ階層にスクロールし、移動します。
    ② Mobile - 下位要素(ナビゲーション)が上位要素の上に拡張されます。

    COMPONENTS REFLOW

    UIは機器別にReflowされるよう、レイアウトを変更します。
    Desktopの水平リストは Mobile画面で垂直リストにReflowされます。

    COMPONENTS TRANSITIONS

    探索転換はユーザーがホーム画面からコンテンツ画面に移動するなど、 画面を移動する際に発生します。
    ① Desktop - Ajaxポップアップによって下位要素(Components)が上位要素の上にポップアップされます。
    ② Mobile - 下位要素(Components)が上位要素の上に拡張されます。
